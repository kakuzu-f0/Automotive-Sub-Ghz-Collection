# Automotive-Sub-Ghz-
A Collection of Various Automotive Sub-Ghz Key Fobs for Research Purposes Only!

## Credits
- **Original Author:** [mrcap665](https://github.com/mrcap665/Automotive-Sub-Ghz-) - Original collection and research
- **Organization & Structure:** [Kakuzu](https://github.com/kakuzu-f0) - File organization, naming standardization, and brand categorization

FILE KEY:
l = lock /
u = Unlock /
rs = Remote Start /
t = trunk /
p = panic button /
sd = side door (auto sliding doors)

The number denotes times pressed (usually twice per button)

## Collection Structure
This collection is organized by automotive brands with consistent `Brand_Model_ButtonCodes` naming format:

- **20 Brand Categories:** All major automotive manufacturers
- **167 Files:** Comprehensive Sub-Ghz key fob collection
- **Standardized Naming:** `Brand_Model_ButtonCodes.sub` format
- **Research Ready:** Optimized for Flipper Zero research purposes

I encounter newer cars daily. Also if there is a better/preferred format for collection please let me know.
